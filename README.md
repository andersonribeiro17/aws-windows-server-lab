# Laboratório AWS EC2 + Windows Server

Projeto realizado em laboratório online para entrega de atividade do curso de Redes de Computadores do SENAI.

## Objetivo
Realizar a criação e configuração de uma infraestrutura Windows Server na AWS utilizando armazenamento EBS, RAID, usuários, grupos e permissões NTFS.

## Atividades realizadas
- Criação de instância EC2 Windows Server
- Configuração de volumes EBS
- Inicialização e gerenciamento de discos
- Implementação de RAID 5
- Implementação de RAID 1
- Criação de usuários locais
- Criação de grupos de usuários
- Associação de usuários aos grupos
- Criação de compartilhamentos de rede
- Configuração de permissões NTFS
- Testes de acesso e restrição entre setores
