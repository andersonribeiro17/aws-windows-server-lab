# Laboratório AWS EC2 + Windows Server

Projeto prático realizado em laboratório online para atividade acadêmica do curso de Redes de Computadores do SENAI.

---

# Objetivo

Implementar e configurar uma infraestrutura Windows Server na AWS utilizando serviços EC2, armazenamento EBS, RAID, gerenciamento de usuários, grupos e permissões NTFS.

---

# Atividades Realizadas

* Criação de instância EC2 Windows Server
* Configuração e gerenciamento de volumes EBS
* Inicialização e administração de discos
* Implementação de RAID 1
* Implementação de RAID 5
* Criação e gerenciamento de usuários locais
* Criação de grupos de usuários
* Associação de usuários aos grupos
* Configuração de compartilhamentos de rede
* Configuração de permissões NTFS
* Testes de acesso e restrições entre setores

---

# Tecnologias Utilizadas

* AWS EC2
* Windows Server
* Amazon EBS
* RAID 1
* RAID 5
* NTFS
* Gerenciamento de usuários e grupos
